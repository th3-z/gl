# NIT3DYNE

A low fidelity 3D engine, OpenGL 3.2.

## Features

- Animation and skinning
- Spot and directional lights
- Cubemaps
- Heightmap terrain
- Virtual resolution
- Post FX
- Per-vertex shading
- Materials
- Affine texture mapping
